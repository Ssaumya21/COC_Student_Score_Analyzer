# COC_Student_Score_Analyzer

# About the Project

A C program for managing and analyzing student scores for a small class. It takes number of students and marks of students as an input from the user, analyze it and gives the average score, Highest, and Lowest marks scored by the students as an output.

# Concepts Used :

## C :-
- 1D Arrays
- For loops
- Functions
- If statements
- Basic Variables and Operators

## Maths :-
- Mean(Average)
- Maximum
- Minimum

# Compile :

- gcc Student_score_Analyzer.c
- ./a.exe

 # How to Run

```
Enter Number of Students: 5
Enter Marks of Student 1: 21
Enter Marks of Student 2: 34
Enter Marks of Student 3: 65
Enter Marks of Student 4: 43
Enter Marks of Student 5: 88
```
